# Audio Folder

Place your background music here:

## Required:
- `music.mp3` - Background music file

## Important Notes:
- Music will NOT autoplay (user must click play button)
- Use royalty-free music to avoid copyright issues
- Recommended file size: 2-5MB
- Recommended bitrate: 128kbps for smaller file size

## Free Music Sources:
- https://pixabay.com/music/
- https://www.bensound.com/
- https://freemusicarchive.org/
